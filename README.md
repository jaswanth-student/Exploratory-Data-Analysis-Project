# Exploratory Data Analysis (EDA) Project

## Objective
Analyze student performance data to uncover patterns and trends.

## Dataset
Students Performance Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Statistical Summary
- Histograms
- Count Plots
- Box Plots
- Correlation Heatmap

## Key Insights
- Reading and writing scores are highly correlated.
- Test preparation course positively impacts performance.
- Female students generally perform better in reading and writing.
